## Overview
LearnLab is a modern, responsive frontend dashboard built for a seamless student experience. It features a scalable architecture utilizing Next.js Server Components for secure data fetching and Framer Motion for high-performance, physics-based UI interactions.

## 🚀 Tech Stack
* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS v4
* **Animations:** Framer Motion
* **Database / Backend:** Supabase (PostgreSQL)
* **Icons:** Lucide React
* **Deployment:** Vercel

## ✨ Key Features
* **Server-Side Data Fetching:** Utilizes Next.js Server Components to securely fetch data from Supabase, keeping database credentials completely hidden from the client.
* **Progressive UI/UX:** Features a Next.js `loading.tsx` skeleton state for instant perceived performance during asynchronous database queries.
* **Physics-Based Animations:** Replaced standard CSS transitions with Framer Motion spring physics for tactile, zero-layout-shift hover states.
* **Fully Responsive Architecture:** Implements a custom CSS Grid (Bento Box style) that seamlessly adapts from desktop to a dedicated mobile layout featuring a glassmorphism navigation overlay.
* **Design System:** Utilizes strict CSS variables for a consistent, premium "Deep Space & Neon" visual aesthetic.
